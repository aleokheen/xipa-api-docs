Get categories

GET /tasks/categories

Filters (GET parameters):

Field Description Required
limit 1-20 false
offset Offset DB query false
searchQuery Query for search by categories or tasks name false

Note that no autorization required for this method

Field Description

Rate limit

10 queries per second (send requests as rarely as possible)
