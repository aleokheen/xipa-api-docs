Joining users

There are POST request which allows everyone to subscribe XIPA news over E-Mail.

If request is success, the letter will be dropped to info@xipa.ru

Required data

field description required
firstName User`s firstname true
email User`s email true
comment Comment. Min 8 symbols false

URL

POST /join

Response

You will receive HTTP status code 201

Limits

3 requests per hour
