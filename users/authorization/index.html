Authorization

Each sensitive request has to contain accessToken parameter. If it`s need to get accessToken for a user, you need:

Required data

field descrption
username User`s username
password User`s password

URL

/accessToken

Response

You will receive json with accessToken field

Limits

3 requests per 15 minutes
