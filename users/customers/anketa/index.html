Filling the customer`s anketa

Send PUT request with required data. You can update anketa info at any time

Required data

Field Description Required
legalName Полное имя юридического лица true
inn ИНН true
ogrn ОГРН (либо ОГРНИП, если ИПшник) true
kpp КПП (у ИПшников его нет) false
bik БИК true
bankName Имя банка true
bankAccount Номер расчётного счёта true
bankKorrAccount Банковский корр. счёт true
lawAddress Юридический адрес true
zipAddress Почтовый адрес true
legalFIO ФИО ответственного лица true
legalPosition Должность ответственного лица true
legalPurpose Основание ответственного лица true
accessToken User`s access token true

URL

/users/customer/anketa

Response

You will receive HTTP status code 201

Limits

10 requests per hour
