Password reset

To reset an account password, it`s needed to send request (near below) and receive resetion code over user`s E-Mail. Then you need send second request (far below)

Getting the code

Required data

Field Descrption
email User`s email

URL

POST /resetPassword

Response

201

{
  "accessToken": "rkegorhhr..."
}

Send resetion request

Required data

field descrption
email User`s email
code Password resetion code
password New password

URL

PUT /resetPassword

Response

201

Limits

5 requests per 30 minutes
