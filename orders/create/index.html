Create an order

Only users with role "customer" can make orders.

To create order, send

POST /order

with parameters

Field Description Required Type
name Order name true String
description Order description true String
completeDate Timestamp (only future) true String or integer
tasks[] Array of below (objects) true Array of objects
tasks[].categoryID Category ID like "602bfe6cf8d293fbec82f93e" true MongoDB ObjectId
tasks[].ids[] Array of indexes of tasks array of a category true Array of ints
tasks[].needMaterial Is it needed materials to do the tasks true Boolean
tasks[].isHeavyTask Is the task heavy true Boolean
address Order address true String
phone Customer`s phone true String
isEmergency Is task urgent true Boolean
accessToken User`s accessToken true String

Request example

{
  "name": "Настроить принте",
  "description": "Шла Саша по шоссе и сосала сушку",
  "completeDate": "2021-02-16T22:54:18+03:00",
  "tasks": [{
    "categoryID": "602bfe6cf8d293fbec82f93e",
    "ids": [0,1],
    "needMaterial": false,
    "isHeavyTask": false
  }],
  "address": "Россия, Москва, ул. Сивицкого, д. 13",
  "phone": "+79999999999",
  "isEmergency": false,
  "accessToken": "c31d3cecf61f412c24160b9aa304b7af8128ad9495b828db1f1509dced31508f"
}

Response

201

JSON with orderID like

{
  "orderID": "602c25b3ff4c52b2198b07b8"
}

Limits

3 requests per 15 minutes
