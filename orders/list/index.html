List orders

To see all XIPA orders, don`t send accessToken. To see only orders created or performed by user, send accessToken.

Request

GET /orders

Filters (GET parameters):

Field Description Required
limit 1-20 false
offset Offset DB query false
searchQuery Query for search by categories or tasks name false

Response

200
